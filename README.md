# Web3新手村 - Solidity 3月共学营预习指南

## 🎯 项目背景
针对3月高强度Solidity共学营设计，帮助0基础学员提前建立必要认知、完成环境配置、消除学习恐惧，提高共学成功率。

## 🚀 快速开始
1. 用浏览器访问[Web3新手村预习指南](https://foroneice.github.io/web3-novice-village/) 
2. 从Level 0开始，按顺序完成所有关卡

## 🎮 关卡系统
| 关卡 | 名称 | 目标 | 预计耗时 |
|------|------|------|----------|
| Level 0 | 环境准备 | 完成所有必要工具注册 | 1-2天 |
| Level 1 | 破除迷雾 | 建立区块链基本认知 | 2-3天 |
| Level 2 | 魔法沙盒 | 体验链上交易和Gas | 1周 |
| Level 3 | 逻辑拼图 | 学习编程基础逻辑 | 1周 |
| Level 4 | 初见 Solidity | 完成第一个合约修改 | 剩余时间 |

## 🌟 特色功能
1. **闯关式学习** - 必须完成前一级才能解锁下一级
2. **环境配置引导** - 特别强调GitHub/X/Telegram等工具注册
3. **模拟交互** - 模拟钱包创建、交易发送等关键操作
4. **进度追踪** - 可视化进度条和成就系统
5. **心理建设** - 专门针对0基础学员的心理指导

## 📚 包含：
• 交互式学习挑战 • 精选资源聚合 • GitHub提交模拟
• Build in Public 指南 • 进度追踪系统 • 心理建设建议

## 🎮 核心理念：预习不是学代码，而是建立认知、消除恐惧
